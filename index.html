Projects

Storyverse

An immersive reading companion for kids powered by GAI. Using RAG (Retrieval-Augmented Generation) Storyverse constructs immersive and age appropriate stories for children learning to read, as well as quizzes to test thier comprehension.

Liquid Legends

An Enter The Gungeon inspired shoot 'em up developed in Unity. The bulk of this project consisted of tweaking controllers and physics scripts in C# for the shooting and movement mechanics. The soundtrack and pixel art graphics were created by me, which was a fun challenge.
